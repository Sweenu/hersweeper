# hersweeper
A pretty neat command line minesweeper

### Installation
To install hersweeper, you need to have pip installed and then:
```
git clone https://github.com/Sweenu/hersweeper
cd hersweeper
make init
```

### Usage
`python -m hersweeper`

### Development
To run the whole test suite (well, it's like 2 tests ^^):

`make test`
